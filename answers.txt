Q1: Why might we consider using Git?

Answer: Git is a useful version control system that makes it easy to view past and present versions of code. This is especially helpful for large, long-term projects. It can also make it easy for team members to collaborate.


Q2: What field could data science be applied to? Give an Example.

Answer: One field is healthcare. Examples within the field include diagnostics, treatment suggestions, and prediction of prognosis.


Q3: What is the difference between Merging and Rebasing?

Answer: Merging requires you to resolve all branch conflicts, while rebasing creates new commits for each commit in the original branch.
